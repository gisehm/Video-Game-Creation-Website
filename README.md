# Video-Game-Creation-Website
A collaborative UI Design project website meant to introduce people to the world of video game creation. The video game industry is constantly growing and, as such, this website offers an easy tutorial for those who want to learn how to create and deploy a basic video game feature! 

## Problem Statement

Video games are a combination of art, music, storytelling, and technology; however, the work behind them is not often spoken about. This makes the video game industry one that is very difficult to break into, which is an issue we hope to reveal and address with this website. In contrast to most tutorials, this one assumes no prior experience and offers interactive quizzes to ensure the user understands the major aspects of video game creation.

## Key Results

1. Upon multiple tests on users, we found that the inclusion of quizzes proved to be very effective as users who took these quizzes showed better understanding than those who didn't.
2. By utilizing images, users were better able to understand the creation process as opposed to a purely written tutorial.
3. The use of shorter sentences and more images allowed the user to better follow the tutorial.

## Methodologies

To accomplish this, we first created a welcoming layout and homepage for the user to easily navigate to the tutorial. For the tutorial and quiz pages, we created templates to hold the images and information, ensuring that the written explanation was succinct; we also included feedback to ensure the user understands any possible mistakes made on the quiz, allowing them to fully understand their errors and improving their learning experience as a result. By utilizing Python, Flask, and jQuery, we were able to create a human-centered website that uses interactive quizzes to simplify a process that is oftentimes confusing.
